# Volatus Aerospace — Design Guidelines
# Developed by SHASHANK DAKSH

## 1. Design Approach
The structure is inspired by leading aerospace and advanced technology companies. The goal is to present a premium, high-precision brand identity supported by clean layouts, smooth motion, and strong visual hierarchy.

## 2. Core Principles
- Premium aerospace aesthetic
- Subtle motion and depth to reinforce innovation
- Clear information flow for complex services
- Consistent, professional UI across all pages

## 3. Typography
Primary: Inter (600–700)  
Secondary: Roboto (400–500)  
Accent: Orbitron (700)

Hierarchy:
- Hero: 56px
- Section Titles: 40px
- Subsections: 28px
- Body: 16px
- Captions: 14px

## 4. Layout System
- 12-column desktop grid (max-width 1440px)
- Tailwind spacing scale (4–32)
- Full-height hero sections, wide content areas, balanced whitespace

## 5. Video Integration
- Hero uses continuous aerial/aerospace footage with a dark gradient overlay
- Secondary pages feature technology or operations footage
- Mobile devices use optimized static images

## 6. Component Structure

### Header Navigation
- Sticky header with blur effect on scroll
- Clear categories: Services, Industries, Training, Equipment, Aviation, About, Contact
- Mobile: Full-screen overlay menu with smooth transitions

### Hero Sections
- Full-viewport layout
- Strong headline, supporting text, dual CTAs

### Service Cards
- Three-column layout
- Hover elevation and “Learn More” action

### Industry Showcase
- Horizontal carousel with imagery and key stats

### Statistics
- Animated count-up metrics for credibility and scale

### Forms
- Multi-step request form with progress indicators

### Footer
- Three columns: links, company info, contact
- Bottom bar: Privacy/Terms + “Developed by SHASHANK DAKSH”

## 7. Animations
- Fade-in and slide-up effects on scroll
- Parallax backgrounds on hero sections
- Page fade transitions
- Buttons and cards increase brightness/elevation on hover

## 8. Accessibility
- WCAG AA color contrast
- Keyboard-accessible navigation
- ARIA labels for interactive components
- Visible focus outlines

## 9. Imagery
- WebP images with PNG fallbacks
- Industry-specific photography across service pages
- Professional operator and equipment images on technical pages

## 10. Additional Features
- Floating “Get Quote” button
- Optional chat widget
- Cookie banner with clean layout
- Search overlay with suggestions
- Custom 404 page with guided navigation
- “Developed by SHASHANK DAKSH” included in footer
